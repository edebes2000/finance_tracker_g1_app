# Finance Tracker Streamlit App

This Streamlit application demonstrates the usage of the FinTrackPy package.

The app allows users to upload a Santander bank statement and generate a financial summary.

## Installation

Install dependencies:

uv sync

## Run the application

uv run streamlit run app.py

## Features

- Upload Excel or CSV bank statement
- Calculate total income
- Calculate total expenses
- Show first 20 transactions
- Visualize spending by category
